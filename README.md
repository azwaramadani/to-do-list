# 📝 Soroban To-Do List Smart Contract

## 📌 Description
This project is a simple **To-Do List decentralized application (dApp)** built using **Soroban Smart Contracts** on the Stellar network.

The application allows users to manage tasks directly on the blockchain, ensuring that data is **transparent, immutable, and decentralized**.

---

## 🚀 Features

- ✅ Create a new task
- 📋 View all tasks
- ✔️ Mark task as completed
- ❌ Delete task
- 🔐 Data stored securely on blockchain

---

## 🧠 How It Works

Each task contains:
- `id` → Unique identifier (generated using PRNG)
- `title` → Task description
- `completed` → Task status (true/false)

All tasks are stored inside the smart contract storage.

---

## ⚙️ Smart Contract Functions

### 1. get_tasks
Retrieve all tasks stored in the contract.

### 2. create_task
Create a new task by providing a title.

### 3. complete_task
Mark a task as completed using its ID.

### 4. delete_task
Delete a task using its ID.
